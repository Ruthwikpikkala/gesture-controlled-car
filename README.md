# Gesture Controlled Car

This project demonstrates a gesture-controlled robotic car using an MPU6050 sensor, NodeMCU, Arduino UNO, and ESP-NOW communication.

## Features
- Real-time gesture control
- ESP-NOW wireless communication
- Low latency response
- Arduino-based motor control

## Hardware Used
- Arduino UNO
- NodeMCU (ESP8266)
- MPU6050
- L298N Motor Driver
- DC Motors

## Software
- Arduino IDE
- ESP-NOW Protocol
